Windows install guide

Install the RxPulse desktop beta safely.

This guide is for approved doctors. The code-signing certificate is intentionally deferred for budget reasons, so a Windows SmartScreen warning may appear. Get the installer only from the official RxPulse support/download channel.

Step 1

Verify file name and checksum

  • File name should be: RxPulse-Clinician-Suite-1.2.0-Setup-x64.exe
  • Match the SHA-256 value with the value provided by support or SHA256SUMS.txt.
  • If the checksum does not match, do not run the installer; contact support.
Step 2

If Windows SmartScreen appears

  • You may see 'Windows protected your PC'.
  • First verify the file name and checksum.
  • Then click More info.
  • Do not panic if the publisher is unknown; this beta build is unsigned.
Step 3

Choose Run anyway and install

  • After clicking More info, Windows shows the Run anyway button.
  • Click Run anyway only if the installer came from the official source and the checksum matches.
  • After installation, open RxPulse from the Start Menu or desktop shortcut.
Data safety

Before update or reinstall

  • Before updating or reinstalling, confirm Cloud Sync Complete.
  • Use one doctor account per Windows user/device workspace.
  • Hide patient identity before sending screenshots or support files.
  • For install or login problems, email support@rxpulsecs.com with Windows version, app version, and screenshot.
