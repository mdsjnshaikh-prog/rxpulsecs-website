Workflow

A simple path from account creation to approved clinical use.

RxPulse separates public account creation from professional approval. A doctor starts with email verification, then completes professional details inside the app so Admin can review identity, BMDC information, and account readiness before access is enabled.

1

Create account

The public signup page asks only for the essentials: email and password. Professional information is not collected here, which keeps onboarding simple and reduces mistakes at the first step.

2

Verify email

The doctor opens the latest verification email and confirms the account within 5 minutes. This confirms ownership of the email before the software login and approval request.

3

Login to RxPulse app

After verification, the doctor opens the RxPulse app or desktop release and logs in with the same credentials used during signup.

4

Complete profile once

Inside the app, the doctor enters BMDC number, professional name, degree, specialty, chamber information, and other approval details only once before submission.

5

Save & Apply

Save & Apply stores the profile and sends it for Admin review. Keeping these details controlled helps prevent accidental changes after the approval request is submitted.

6

Admin review

Admin reviews the submitted profile and can approve access, approve with trial, reject, disable, or request correction according to account status and RxPulse policy.
