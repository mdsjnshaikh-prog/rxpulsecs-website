Desktop install issues

  • Download the installer only from the official download page or GitHub Release.
  • If Windows SmartScreen appears, verify the file name and checksum. Code signing is still pending.
  • Internet is required for first login and cloud sync.
  • Before uninstalling or reinstalling, confirm Cloud Sync Complete.
🔄

Sync and offline help

  • After prior login, work can continue during weak internet.
  • If unsynced data exists, check the warning before closing or signing out.
  • Use one doctor account per Windows user/device workspace.
  • For sync problems, include sync status, error message, and internet condition.

Before sending screenshots or backup files to support, hide patient name, phone, address, or anything that identifies a patient. Active local data protection is improved, but use a secure Windows login/profile.
