RxPulse desktop beta access and safe install guidance.

The Windows desktop beta is distributed through controlled access for approved doctors. The installer link is not public; request access from your registered email.

Windows Desktop Beta

RxPulse Clinician Suite 1.2.0

The file name should be RxPulse-Clinician-Suite-1.2.0-Setup-x64.exe. The final SHA-256 checksum will be provided with the verified build through the support/download channel.

Current status

  • Access mode: approved doctors by request
  • Code signing: planned for a future release due to budget
  • Auto-update: planned for a future release; initial beta uses manual update
  • WebApp version: 1.2.0
Checksum

Verify the installer

  • Get the installer only through the official RxPulse support/download channel.
  • Match the exact SHA-256 value or SHA256SUMS.txt provided through the support/download channel.
  • If the checksum does not match, do not run the installer; contact support.
SmartScreen

A warning is expected

The code-signing certificate is intentionally deferred for budget reasons, so Windows SmartScreen may show a warning. After verifying file name and checksum, follow the install guide.

Windows install guide
Install notes

Before installing

  • Internet is required for first login and cloud sync.
  • After prior login, chamber work can continue during weak internet.
  • Sync before uninstalling, reinstalling, or switching devices.
  • Use one doctor account per Windows user/device workspace.
