Checksum

Verify the installer

Each desktop rebuild can have a different SHA-256 checksum. When you receive the installer, match it with the exact SHA-256 value or SHA256SUMS.txt provided through the support/download channel.

If the checksum does not match, do not run the installer. Contact support for a fresh verified link.

Install notes

Before installing

  • Internet is required for first login and cloud sync.
  • After prior login, chamber work can continue during weak internet.
  • Sync before uninstalling, reinstalling, or switching devices.
  • Use one doctor account per Windows user/device workspace.
  • Code signing is deferred for budget reasons, so Windows SmartScreen may show a warning.
Unsigned installer guide

What to do if SmartScreen appears

  • First verify the file name and SHA-256 checksum.
  • If Windows protected your PC appears, click More info.
  • Then click Run anyway — only if the source is official and the checksum matches.

Support and data safety

If you send screenshots or backups to support, hide patient name, phone, address, or anything that identifies a patient. The desktop app improves local protection, but a secure Windows login/profile is still important.
